# Code of Conduct

## The short version

Be respectful, be constructive, be professional. We're here to build better engineering tools together.

## Standards

- **Do**: Give constructive feedback, assume good intent, help others learn, focus on the work
- **Don't**: Harass, insult, troll, or publish others' private information

## Enforcement

Unacceptable behavior can be reported to **support@renozoic.com**. Reports will be reviewed and acted on. Repeated or severe violations may result in removal from the project.

## Scope

This applies to all project spaces — issues, PRs, discussions, and any context where you're representing this project.
